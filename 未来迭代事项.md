# 🚀 eApp 未来迭代事项规划

本文件用于记录应用后续的版本迭代规划、新功能想法以及待优化的技术架构与用户反馈问题。

---

## 📅 待办与规划事项

### 1. ☁️ 基于云端自动更新升级 (Updater) 机制 【高优先级】
- **需求描述**：实现应用内基于云端的一键自动更新升级与静默升级，让用户无需手动去 GitHub 下载新版本包。
- **技术实现大纲**：
  - **数字签名**：在本地运行 `npx tauri signer generate` 生成公私钥对，并将公钥硬编码写入 `tauri.conf.json` 中，私钥安全地配置到 GitHub 仓库的 Secrets 中（命名为 `TAURI_SIGNING_PRIVATE_KEY`）。
  - **依赖安装**：在 Rust 端引入 `tauri-plugin-updater` 插件，并在前端 Vue 应用中调用 `@tauri-apps/plugin-updater` 的 `check()` 与 `downloadAndInstall()` API。
  - **托管方案**：使用 GitHub Actions 在发布新 release 时自动调用私钥进行签名，生成包签名文件，并更新项目根目录下的 `updater-info.json`。客户端以此 JSON 作为静态更新 API 地址。
- **参考配置文件 (`tauri.conf.json`) 预设**：
  ```json
  "plugins": {
    "updater": {
      "active": true,
      "pubkey": "YOUR_GENERATED_PUBLIC_KEY",
      "endpoints": [
        "https://raw.githubusercontent.com/ByronYan01/eApp/main/updater-info.json"
      ]
    }
  }
  ```

### 2. 🐞 Bug 修复与基础交互优化 【高优先级】
- **播放倍速不生效** 🔴
  - **功能描述**：当前音频播放器的倍速设置无法生效，需要修复音频播放组件的速率调节机制。
- **快速清空输入框内容** 🟡
  - **功能描述**：在核心输入框中增加“清空”按钮（如一键清除的 ✖ 图标），便于用户快速重置输入框内容，提升输入效率。
- **整句连读音标去重与净化** 🟡
  - **功能描述**：当前整句连读音标解析中，部分单词会返回两个音标（例如 `I’m new. I just moved here from` 中的 `from` 会返回两个音标）。需要优化音标解析算法或提取逻辑，进行去重或优先选择合适音标展示。

---

### 3. 🃏 卡片复习与句子仓库功能增强 【中优先级】
- **卡片复习随机模式优化（听音辨意/盲听模式）** 🟢
  - **功能描述**：
    1. 支持随机模式，且默认先播放句子的声音，不展示文字内容。
    2. 如果用户没听清，允许点击“再次播放”重复收听音频。
    3. 用户可以选择“看原句”，此时再展示该句子的智能解析结果（作为反馈参考）。
- **详情页集成智能解析** 🟢
  - **功能描述**：在“卡片复习”和“句子仓库”中，当用户点击具体每个句子的详情时，详情页面应该能够展示出该句子的**智能解析结果**（即与主查询页相同级别的语法、连读发音、词义解析等深度分析内容）。
- **生词本导出**
  - **功能描述**：支持将学习列表中的句子与单词一键导出为 PDF、Excel 或 Anki 卡片包格式，方便多软件协同复习。

---

### 4. ⚙️ 系统设置与接口优化 【中优先级】
- **接口测速功能** 🔵
  - **功能描述**：在默认设置（Settings）中，增加一个在线 API 接口测速功能，方便用户查看不同节点的连接延迟。
- **快速应用在线 API 最佳配置** 🔵
  - **功能描述**：在设置界面提供一键“快速应用在线 API 最佳配置”的选项，系统自动测速并应用表现最出色的 API 设置组合。
- **离线音效/本地发音包缓存**
  - **功能描述**：为了保证在断网或网络极差的环境下仍能发音，开发本地 TTS 音频缓存功能，避免重复请求 API。
- **精美可视化仪表盘**
  - **功能描述**：增加学习统计图表（折线图/柱状图），展示每日学习句子的数量、复习记忆率等数据，激励用户坚持学习。

---

### 5. 🔄 跨设备同步 (Sync) 机制 【低优先级】
- **功能描述**：用户在多台电脑（如办公室 Windows 和家里 Mac）上学习时，学习进度和生词列表能自动同步。
- **技术方案**：支持绑定 WebDAV（如坚果云）或接入 Supabase/Cloudflare KV 等免费数据库，让用户无需注册账号即可利用个人云盘进行加密数据同步。
