# 🚀 eApp 迭代事项与版本路线规划

本文件用于实时记录客户端的功能迭代规划、技术架构调优、用户体验反馈，以及已发布版本的历史成果归档。

---

## 📅 未来待办与规划事项 (Roadmap)

### 1. 💾 在线发音本地离线缓存 (Offline Audio Cache) 【中高优先级】
- **功能描述**：当在线原声发音下载成功后，在本地以文件的形式缓存在用户的应用目录中。下一次播放同一个句子或单词的发音时，如果本地已缓存，则直接播放本地文件，不再进行网络请求。
- **技术要点**：
  - 在 Rust 后端下载成功时，自动将音频文件以其文本 MD5 命名的 `.mp3` 写入本地 AppData 的缓存子目录中。
  - 请求发音时，优先检查本地是否存在该缓存文件。若存在则直接读取文件返回 Base64，否则才发起网络请求。
  - 在设置页面中增加“发音缓存大小展示”与“一键清理缓存”的按钮。

### 2. 📤 单词/生词本多格式导出 (Data Export) 【中优先级】
- **功能描述**：支持将收藏的句子和生词导出，方便多软件协同复习或打印阅读。
- **技术要点**：
  - **Excel / CSV 导出**：导出包含“英文原句、中文翻译、整句音标、单词拆解及释义”的表格文件，方便打印背诵。
  - **Anki 卡片包导出**：导出为 Tab 分隔的文本格式，直接契合 Anki 导入规范，供用户在手机端 Anki 离线背诵。

### 3. 📊 学习数据可视化统计仪表盘 (Dashboard) 【中低优先级】
- **功能描述**：提供炫酷的学习反馈，增加学习成就感，刺激用户坚持复习。
- **技术要点**：
  - 新增“学习统计”导航 Tab。
  - 引入轻量级图表库，用半透明发光的曲线展现：每日新增收藏句子走势、艾宾浩斯复习熟练度占比、每日复习完成度打卡。

### 4. 🔄 冷启动静默检查更新 【低优先级】
- **功能描述**：当前版本检查必须进入设置手动点击。可优化为客户端每次冷启动时，在后台线程静默读取云端发布信息，一旦检测到有比当前更高的新版本时，主动在界面右下角弹出气泡轻量提醒。

### 5. 🎨 界面主题与字号微调 【低优先级】
- **功能描述**：支持手动切换深色星空模式 (Dark) / 亮色磨砂玻璃模式 (Light)；并提供全局单词查词框的字体大小和音标显示字号大小调节。

## 🎉 已完成的开发成果归档 (Changelog)

### v0.1.1 (当前版本)
*   **🔄 跨设备 GitHub Gist 数据同步**：
    *   在 Rust 后端实现 Gist 网络请求网关，完美绕过前端网络跨域限制。
    *   实现了无损的进度合并策略：同 ID 难句发生冲突时，智能比对两端的复习次数（`reviewCount`），保留记忆熟练度更高的记录，防止进度倒退。
    *   提供“一键自动在云端新建私人 Gist”功能，回填保存 Gist ID。
    *   在同步设置界面增加了可平滑展开折叠的“极简 4 步配置使用指南折叠面板”及直达链接。
*   **⚙️ 翻译/发音网络请求超时调节**：
    *   重构 reqwest HTTP 客户端，增加对每次调用动态设定超时限制的功能，避免网络连接长时间死挂。
    *   在系统设置中增加了“整句翻译超时”和“在线发音超时”两个高颜值滑动条控件（1000ms - 10000ms）。
    *   测速中心在测速时默认应用 3000ms 超时限制，防止卡死。
*   **🌐 引入高可用 MyMemory 翻译接口**：
    *   针对有道、百度免签接口失效导致智能解析报错“翻译获取失败”问题，引入了在国内外均直连畅通的 MyMemory 翻译源。
    *   将 MyMemory 作为“智能兜底 (Auto)”的默认第一顺位翻译服务。

### v0.1.0 (基础版本)
*   **🔊 发音倍速与单词语速一致性控制**：
    *   攻克了在线真人发音对播放速率控制（`playbackRate`）的绑定支持。
    *   主查询界面及卡片复习界面中点击单个单词的发音语速，由硬编码的 `1.0` 调整为绑定用户全局设置的倍速。
*   **✖️ 输入框快速清空优化**：
    *   增加清空按钮点击响应热区，绑定 `Esc` 键盘快捷键清空并重聚焦文本框。
*   **⚡ API 接口延迟测速与最佳配置应用**：
    *   在设置卡片底部新增「API 测速中心」，通过后端划词代理并发检测各接口延迟。
    *   支持一键「应用最佳接口配置」，自动将翻译和在线发音源切为最快的可用提供商。
*   **🃏 卡片复习随机模式与盲听交互**：
    *   支持“顺序/随机”复习切换。使用 Fisher-Yates 算法在内存中维护打乱的映射队列。
    *   在答案未揭晓前以动态雷达声波的 `🎧` 图标替代文本原句，引导用户“盲听”回想。
    *   智能解析整合：揭晓后平滑淡入呈现难句智能解析详情（悬浮查词、释义词典等）。
*   **☁️ Tauri Updater 自动更新集成**：
    *   在 Rust 端集成 `tauri-plugin-updater`。
    *   授权 Capabilities 沙箱权限，解决了 check 拦截错误。
    *   前端集成高颜值更新升级确认弹窗。
